# SkyPath — Flight Connection Search Engine

A prototype flight connection search engine for a travel startup. Given a dataset of flight schedules, users should be able to search for valid flight itineraries between two airports, including multi-stop connections.

## Run

```bash
docker-compose up
```

## Status

Skeleton in place

## Error handling

Dealing with data issues - "JKF" to "JFK", prices with string format.
